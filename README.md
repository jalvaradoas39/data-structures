# Linear Search

	* What is linear seach?
		a process that checks every element in the list sequentially until the desired element is found.
	* When use it?
		1. list is short
		2. unordered
	* BigO(worst and best case)
		worst case: O(n)
		best case: O(1)
	* JavaScript functions using some sort of linear search
		1. indexOf
		2. includes
		3. find
		4. findIndex
	* How use it?
		solution
		callstack (view)
